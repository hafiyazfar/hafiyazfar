## 🚀 About Me

I bridge **clean code** and **real-world impact** — with a focus on how AI and IoT ecosystems can tackle critical problems in sustainability and resource management.

- 🛠️ **Building:** Smart mobile apps with Flutter/Dart — green-tech solutions, and intuitive UI/UX
- 🌱 **Exploring:** Computer vision for disease detection, IoT sensor integration, and scalable cloud infrastructure on AWS
- ⚡ **Belief:** If the UI/UX isn't clean, the backend logic never gets to shine
